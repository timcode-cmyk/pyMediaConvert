## 1. 过滤不支持TTS的模型

### 实现方式
在 `on_models_loaded()` 中添加模型过滤逻辑：
```python
# 仅保留支持TTS的模型
tts_models = [m for m in models if m.get('can_do_text_to_speech', False)]
```

### 效果
- ✓ 只显示实际支持文本转语音的模型
- ✓ 避免用户选择不支持TTS的模型而导致错误

## 4. 情绪标签管理器（仅V3模型）

### 核心改动

#### 4.1 新增 EmotionTagManager 类
支持两种方式使用情绪标签：

**方式1：自动检测和插入**
- 点击"🤖 自动检测并插入情绪标签"按钮
- 自动分析文案，按照循环方式为每个句子分配情绪
- 示例输出：
  ```
  [cheerful] 句子1
  [neutral] 句子2
  [hopeful] 句子3
  ```

**方式2：手动拖入**
- 可拖动情绪标签按钮到文本中
- 支持复制-粘贴方式
- 提供6种情绪选择：
  - 😐 中立 (neutral)
  - 😊 欢快 (cheerful)
  - 😢 悲伤 (sad)
  - 😨 害怕 (fearful)
  - 😠 愤怒 (angry)
  - 🤗 希望 (hopeful)

#### 4.2 文本格式
标准格式为：`[emotion_key] 文本内容`

示例：
```
[thoughtful] पतरस ने एक बार कहा था कि वह यीशु को कभी अस्वीकार नहीं करेगा।
परन्तु डर ने उसे तीन बार जकड़ लिया।
[annoyed] "मैं उस व्यक्ति को नहीं जानता।"
जब मुर्गे ने बाँग दी, तो उसे एहसास हुआ कि उसने क्या किया है।
```

#### 4.3 V3模型检测
在 `on_model_changed()` 中检查模型是否为V3：
```python
is_v3 = 'v3' in model_id.lower()
self.emotion_manager_group.setEnabled(is_v3)
```

### 效果
- ✓ 情绪标签功能仅在V3模型时启用
- ✓ 非V3模型时自动禁用情绪组件
- ✓ 灵活的情绪标签插入方式
- ✓ 支持多语言文本中的情绪标签

## 6. 使用流程

### 基本流程
1. **刷新配置** - 点击"🔄 刷新配置"加载模型列表
2. **选择模型** - 从模型下拉菜单选择（自动过滤支持TTS的模型）
3. **选择语言** - 从语言下拉菜单选择（显示中文名称）
4. **输入或编辑文案**
   - 方式A：直接输入文案，然后点击"自动检测并插入情绪标签"
   - 方式B：手动拖入情绪标签按钮到合适位置
5. **生成语音** - 点击"生成语音"按钮

### 情绪标签使用示例

**自动插入后的文案：**
```
[cheerful] 早上好，今天天气真好！
[neutral] 我们来讨论一下这个问题。
[hopeful] 我相信我们一定能成功！
[sad] 但有时候事情并不如我们所愿。
[angry] 这真的很令人沮丧！
```
