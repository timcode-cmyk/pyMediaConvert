# ElevenLabs 功能快速参考

## 🚀 快速开始

### 文本转语音（TTS）

1. **输入 API Key**
   - 在顶部"API 配置"区输入 ElevenLabs API Key
   - 点击"💾 保存"按钮保存到本地
   - 点击"🔄 刷新配置"验证 Key 并加载声音列表

2. **配置语音参数**
   ```
   选择声音模型: [声音列表]  [⚙️ 语音设定]  [🔊 试听]
   
   模型:      [eleven_multilingual_v2 ▼]
   语言:      [中文（简体）▼]
   情绪:      [默认（无特定情绪）▼]
   ```

3. **输入文本并生成**
   - 在文本框输入要转换的内容（支持混合语言）
   - 选择字幕选项（可选：翻译、逐词、XML导出）
   - 点击"生成语音"按钮
   - 生成完成后可在下方播放器中试听

## ⚙️ 语音设定对话框

点击"⚙️ 语音设定"打开对话框，可调整：

### 参数说明

| 参数 | 范围 | 说明 | 建议值 |
|------|------|------|--------|
| **稳定性** | 0-100% | 控制情感变化 | 50% |
| **相似度提升** | 0-100% | 声音与原声相似度 | 75% |
| **风格** | 0-100% | 夸张程度 | 0% |
| **速度** | 0.7-1.2 | 播放速度 | 1.0 |
| **扬声器增强** | 开/关 | 是否增强相似性 | ✓ 开启 |
| **情绪标签** | 6 个选项 | 语调情绪 | 默认 |

### 情绪标签详解

- **😐 中立** → 新闻、教程、技术文档
- **😊 欢快** → 儿童内容、营销、广告
- **😢 悲伤** → 故事叙述、文学作品
- **😨 害怕** → 恐怖、悬念内容
- **😠 愤怒** → 新闻评论、戏剧表演
- **🤗 希望** → 励志内容、鼓励性文案

## 📊 字幕选项

### 快速对比

| 功能 | 说明 | 使用场景 |
|------|------|---------|
| **自动翻译（中）** | 生成中文翻译字幕 | 标准字幕 |
| **逐词字幕** | 按单词/词组分割 | 视频编辑、字幕编辑 |
| **导出 XML** | FCPXML 格式 | Final Cut Pro/DaVinci Resolve |
| **高亮关键词** | 自动标记重要词汇 | 视频创意编辑 |

### 使用流程

```
生成语音 → 自动生成标准字幕 (.srt)
         ↓
         ├→ [逐词字幕] → 按词分割 (_word.srt)
         ├→ [自动翻译] → 中文字幕 (_cn.srt)
         └→ [导出 XML] → 剪辑软件导入 (.fcpxml)
                          └→ [高亮关键词] → AI识别重点词汇
```

## 🎵 音效生成（SFX）

### 使用方法

1. 切换到"🎵 音效生成 (SFX)"标签页
2. 在"描述音效"框输入详细描述
   - ✓ "footsteps on wooden floor" （好）
   - ✓ "heavy rain with thunder" （好）
   - ✗ "rain" （太简洁）

3. 设置时长（1-22 秒）
4. 点击"生成音效"

### 描述技巧

- **详细具体** → "gentle wind blowing through leaves"
- **包含质感** → "metallic, harsh, soft, warm"
- **说明场景** → "small room, large hall, outdoor"
- **参考对比** → "like movie title..., similar to..."

## 💾 配置 (config.toml)

### 简单配置

```toml
[elevenlabs]
api_key = "sk-xxxxxxxxxxxx"
default_model_id = "eleven_multilingual_v2"
default_language_code = "zh-CN"
default_emotion = ""
```

### 高级配置

```toml
[elevenlabs]
# API 相关
api_key = "sk-xxxxxxxxxxxx"
default_output_format = "mp3_44100_192"  # 更高质量

# 模型和语言
default_model_id = "eleven_turbo_v2_5"   # 快速模式
default_language_code = "en-US"          # 英文
default_emotion = "cheerful"             # 欢快

# 字幕切分规则
srt_pause_threshold = 0.15              # 更灵敏的切分
srt_max_chars = 30                       # 更短的行长

# 调试模式
debug_save_response = true               # 保存 API 响应
```

## ❓ 常见问题

### Q: 为什么生成失败？
A: 
- ✓ 检查 API Key 是否正确
- ✓ 检查额度是否充足（点击"🔄 刷新配置"查看）
- ✓ 检查网络连接
- ✓ 查看底部状态栏的错误信息

### Q: 如何选择最合适的模型？
A:
- 标准应用 → `eleven_multilingual_v2` ✓ 推荐
- 英文专用 → `eleven_english_v2` ✓ 推荐
- 实时应用 → `eleven_turbo_v2_5` ✓ 推荐

### Q: 情绪标签有效果吗？
A: 取决于模型和声音是否支持。建议：
- 优先调整"稳定性"参数
- 情绪标签作为辅助选项
- 生成后试听对比效果

### Q: 如何同时生成英文和中文字幕？
A:
1. 勾选"自动翻译（中）"
2. 生成语音，自动生成：
   - 英文字幕 (.srt)
   - 中文翻译字幕 (_cn.srt)

### Q: SFX 音效为什么总是奇怪？
A: 音效生成对描述很敏感：
- ✓ 更详细的描述
- ✓ 包含时间长短信息
- ✓ 提供参考对比
- ✓ 尝试多次生成
