【UI 获取参数】
─────────────────
# 从下拉框获取用户选择
model_id = self.combo_model.currentData()        # 获取模型 ID
language_code = self.combo_language.currentData() # 获取语言代码
emotion = self.combo_emotion.currentData()       # 获取情绪

# 传递给 TTSWorker
self.tts_worker = TTSWorker(
    ...,
    model_id=model_id,
    language_code=language_code,
    emotion=emotion
)


【API 请求体】
──────────────
# 动态构建请求
data = {
    "text": self.text,
    "model_id": self.model_id,                    # ✨ 动态
    "voice_settings": {
        "stability": 0.5,
        "similarity_boost": 0.75,
        "style": 0,
        "use_speaker_boost": True,
        "speed": 1.0,
        "emotion": self.emotion                   # ✨ 动态
    }
}

# 条件性添加语言代码
if self.language_code:
    data["language_code"] = self.language_code    # ✨ 条件添加

🚀 快速开始
═══════════════════════════════════════════════════════════════════════════════

【步骤 1】简单模式（使用默认值）
  1. 输入 API Key 并保存
  2. 刷新配置加载声音列表
  3. 选择声音 → 输入文本 → 点击生成
  ✓ 使用 Config 中的默认模型/语言/情绪

【步骤 2】自定义模式（选择特定参数）
  1. 输入 API Key 并保存
  2. 在 TTS 界面选择:
     • 模型: eleven_english_v2 (英文优化)
     • 语言: en-US (美国英文)
  3. 点击 ⚙️ 语音设定 → 选择情绪: cheerful (欢快)
  4. 输入文本 → 生成语音

【步骤 3】全局配置（编辑 config.toml）
```toml
[elevenlabs]
default_model_id = "eleven_turbo_v2_5"
default_language_code = "en-US"
default_emotion = "cheerful"
```
  ✓ 下次启动时自动应用这些设置
